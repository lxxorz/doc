在 typescript 声明一个变量可以通过显式声明和隐式类型推断

显式类型声明
```ts
const a:boolean = false;
// a is boolean
```

更具体的指定类型
```ts
const a:false = false;
```

隐式类型声明
隐式类型声明略微有点不同，因为使用 `const` 标识符声明变量，变量不可再被赋值，而 let 有更多可能的赋值，所以使用 let 声明的变量有一个更宽泛的类型
```ts
const a = false;
// a is false

let b = false;
// b is boolean
```

更具体的指定类型
```ts
const a = false as const;
```
